# lockfile-tools

Tools to validate and secure your npm ecosystem lockfiles.

# [eslint-plugin-lockfile]

An eslint plugin to validate your npm ecosystem lockfile(s).

# [lintlock]

A standalone CLI that runs the eslint rules with the recommended config.

# [lockfile-tools]

A package of tools shared between other projects in this repo.

[eslint-plugin-lockfile]: https://npmjs.com/eslint-plugin-lockfile
[lintlock]: https://npmjs.com/lintlock
[lockfile-tools]: https://npmjs.com/lockfile-tools
